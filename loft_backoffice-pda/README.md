loft_backoffice-pda
===============
Web APIアプリケーションのアプリケーション層のクラス, 設定ファイル, 単体テストケース等を格納するプロジェクト. 最終的にWebアプリケーションとして*.warファイル化する.
<実行コマンド>

```console
$ java -jar -Xms2048m -Xmx2048m loft_backoffice-pda/target/loft_backoffice-pda-1.0.0-SNAPSHOT.jar --spring.profiles.active=(local or develop or staging or production) &
```
